Mini-página para solicitar TMDB API Key (si te la piden)

Si TMDB te solicita un “website URL”, puedes usar esta misma página local como demo. Sube el contenido de la carpeta frontend/ a GitHub Pages (o a cualquier hosting) y coloca esa URL.

  1. Crea un repo (por ejemplo api-gateway-frontend)
  2. Sube los archivos frontend/index.html, styles.css, app.js
  3. Activa GitHub Pages (Settings → Pages → Deploy from branch)
  4. Usa la URL publicada (por ejemplo https://TUUSUARIO.github.io/api-gateway-frontend/) en TMDB
